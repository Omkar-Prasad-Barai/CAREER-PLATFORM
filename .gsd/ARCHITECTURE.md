## Data Flow

1. **Auth:** User submits login → `authService.login()` → `POST /api/auth/login` → JWT returned → stored in localStorage/sessionStorage → `AuthContext` hydrates state
2. **Protected Routes:** `<ProtectedRoute>` checks `AuthContext.isAuthenticated` + `role` → allows or redirects
3. **API Calls:** Component calls `apiService` → `api.ts` Axios instance attaches JWT → Express route → `authMiddleware` verifies → controller → Mongoose → MongoDB
4. **Connections:** Seeker requests connection → status `pending_admin` → Admin approves/rejects → email sent to seeker → notification created
5. **File Uploads:** Multer middleware → Cloudinary (if configured) or local `/uploads` directory
